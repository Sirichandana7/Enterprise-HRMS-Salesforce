# Business Objective

ABC Global Technologies currently manages employee information using multiple disconnected systems including spreadsheets, emails, and paper-based approvals.

The objective of this project is to centralize HR operations using Salesforce and Agentforce, enabling automation, security, reporting, and AI-powered assistance.

---

# Current Challenges

• Employee data is maintained in Excel.

• Leave approvals happen over email.

• Recruitment tracking is manual.

• Attendance is maintained separately.

• Employee documents are stored on shared drives.

• Reporting requires manual effort.

• No centralized dashboard.

• No AI support for HR operations.

---

# Expected Business Benefits

• Centralized employee records

• Automated recruitment

• Automated leave approvals

• Employee self-service

• AI-powered HR assistant

• Real-time dashboards

• Secure document management

• Enterprise-grade security

• Mobile accessibility

• Reduced manual work
